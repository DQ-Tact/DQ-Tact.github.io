た弱き者の命軽し

Perk
Name Description
た弱き者の命軽し Greatly raises AGL for 3 turns and removes some status ailments when the user's HP drops to 70% or less, 1 time per battle
If the user has not acted this turn, then the user will be the next unit to act, regardless of original turn order
This perk can be triggered when the attack is from an ally
